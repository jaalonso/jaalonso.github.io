En ℝ, |ab| ≤ (a²+b²)/2

José A. Alonso

04-09-2023

Números reales

Sean \(a\) y \(b\) números reales. Demostrar con Lean4 que \[|ab| \leq \frac{a^2 + b^2}{2}\]

Para ello, completar la siguiente teoría de Lean4:

import Mathlib.Data.Real.Basic

variable (a b : ℝ)

example : |a * b| \\leq (a ^ 2 + b ^ 2) / 2 :=
