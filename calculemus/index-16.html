Si (m ∣ n ∧ m ≠ n), entonces (m ∣ n ∧ ¬(n ∣ m))

José A. Alonso

13-12-2023

Números naturales

Demostrar con Lean4 que si (m ∣ n ∧ m ≠ n), entonces (m ∣ n ∧ ¬(n ∣ m)).

Para ello, completar la siguiente teoría de Lean4:

import Mathlib.Data.Nat.GCD.Basic

variable {m n : ℕ}

example
  (h : m ∣ n ∧ m ≠ n)
  : m ∣ n ∧ ¬ n ∣ m :=
by sorry
